01 · 문제 정의

디지털 프라이버시 위기,
지금 당장 해결해야 한다

우리는 매일 수십 개의 약관에 동의하지만, 아무도 그 내용을 읽지 않습니다. 그 사이 개인정보는 조용히 새어나가고 있습니다.

72분

Apple 약관 한 건을 모두 읽는 데 걸리는 시간. 아무도 읽지 않는 이유가 있다.

<9%

실제로 약관을 읽는 사용자 비율. 91%는 내용 모르고 동의한다.

3.5조

연간 전 세계 피싱 피해 규모(원). 점점 정교해지는 스미싱 공격.

수십만

국내 연간 개인정보 침해 신고 건수(2023). 대부분 인지 못한 채 발생.

⚖️

정보 비대칭

기업은 알지만 사용자는 모른다. 법률 용어로 가득한 약관은 의도적으로 이해하기 어렵게 설계된다.

🧠

인지 과부하

평균 수십 페이지에 달하는 약관. 하루에도 여러 번 반복되는 "동의하기" 버튼. 인간의 처리 한계를 넘어선다.

🎣

실시간 피싱 위협

AI로 생성된 정교한 피싱 메시지와 스미싱. URL 하나 클릭으로 모든 개인정보가 유출된다.

📋

무의식적 PII 노출

복사·공유·문서 작성 과정에서 이름, 주민번호, 연락처가 무심코 노출된다. 본인도 모른다.

02 · 비전 & 핵심 가치

Guardian AI가 제시하는
Digital Sovereignty

단순한 앱이 아닙니다. 디지털 세계에서 개인의 주권을 지키는 최후의 방어선입니다.

모든 사람이 약관을 이해할 권리가 있다.
모든 사람이 자신의 데이터를 통제할 권리가 있다.
Guardian AI는 그 권리를 기술로 현실화한다.

🔐

Privacy First

모든 분석은 온디바이스(On-Device) 또는 익명화 처리를 원칙으로 한다. Guardian AI 자신이 또 하나의 감시자가 되지 않는다.

⚡

Clarity Over Complexity

72분짜리 약관을 30초로. 법률 언어를 인간의 언어로. 모든 정보는 누구나 이해할 수 있는 형태로 제공한다.

🛡️

Proactive Defense

위협이 오기를 기다리지 않는다. 실시간으로 감지하고, 사전에 경고하며, 즉각 차단한다.

🤝

User Empowerment

단순히 정보를 제공하는 것을 넘어, 사용자 스스로 올바른 판단을 내릴 수 있도록 힘을 부여한다.

페르소나 설명 핵심 니즈
🧑‍💼 직장인 A 업무 중 다양한 SaaS 서비스에 가입 약관 빠른 요약, 위험 조항 알림
👩‍🎓 대학생 B 앱 설치 시 약관을 무조건 동의 쉬운 이해, 피싱 링크 탐지
👴 시니어 C 스미싱·보이스피싱 피해 취약계층 직관적 UI, 위협 즉시 경고
👩‍💻 개발자 D API·오픈소스 라이선스 검토 필요 상세 분석, 위험 점수 제공

핵심 아키텍처 결정 (ADR)

ADR-0001
모바일 플랫폼: Flutter
1인 개발 + 6주 일정에서 Android/iOS를 동시에 지원하기 위해 단일 코드베이스를 선택.
ADR-0002
상태 관리: setState (Provider는 검토 후 보류)
화면 3개(홈/약관/피싱) 규모의 MVP에서는 화면 간 상태 공유 요구가 거의 없어 setState로 충분. 앱 규모가 커지면 Provider 마이그레이션 경로를 따름.
ADR-0003
백엔드 없이 Flutter + Claude API 직접 호출
사용자 데이터를 서버에 저장하지 않는다는 Privacy First 원칙과 일치. 6주 일정에서 서버 구축·운영 비용 제거.
ADR-0004
인증 없음 (로컬 전용)
로그인 없이 기기 내에서만 동작. 계정 시스템 자체가 공격 표면이 되지 않음.
ADR-0005
배포: APK 사이드로드
스토어 심사 없이 발표 당일 즉시 설치해 시연 가능.
